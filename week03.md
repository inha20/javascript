 # 객체지향 프로그래밍
> 실습

<details><summary>
클래스 만들고 구성요소 채우기
</summary>
➜ id : String 은 String id로.<br>
➜ 앞에 +는 모두 사용 가능 public. <br>
➜ 앞에 ~는 패키지 내부 한정 기본값 default.<br>
➜ 앞에 -는 클래스 내부 한정 private.<br>
➜ 앞에 #는 자식 클래스 한정 protected.<br>
➜ 메소드는 반환타입 ex)void. <br>
➜ 패키지 정보 쓰고나서 클래스에 public 붙여. <br>
➜ src에 만들어서 넣어. <br>
</details>

➜ name의 setter는 setName메소드로, this.name=name. <br>
