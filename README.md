## ✅ 시험 직전 10분 체크리스트

> 시험장에서 이 순서대로 확인하고 들어가라.

1. **UML** → `Exam/UML_To_Java.md`
   - interface / abstract class / extends / implements / super()
2. **interface** → 다중 implements 가능, 변수는 자동 `public static final`
3. **abstract** → 인스턴스화 불가, 추상메소드는 이탤릭체 = abstract 키워드
4. **extends** → 단일상속만 가능, 자식클래스 생성자에서 super() 첫 줄
5. **implements** → interface 구현, 모든 추상메소드 @Override 필수
6. **super()** → 부모 생성자 호출, 자식 생성자 첫 번째 줄
7. **static** → 클래스 소속, 인스턴스 없이 호출 가능. static메소드 → 인스턴스메소드 호출 불가
8. **ArrayList** → `Exam/Student_Template.md`
   - `new ArrayList<Student>()` / `.add()` / `.get()` / `.size()`
9. **BufferedWriter** → `Exam/File_IO_Template.md`
   - `bw.write()` / `bw.newLine()` / try-with-resources
10. **BufferedReader** → `Exam/File_IO_Template.md`
    - `br.readLine()` / while loop / `line.isBlank()` 체크
11. **split()** → `line.split("\\|")` — 파이프는 이스케이프 필요
12. **LocalDate** → `Exam/LocalDate_Template.md`
    - `.now()` / `.of()` / `.parse()` / `.plusDays()` / `DateTimeFormatter`
